# 🎨 watchborne/frontend app

Next.js dashboard to fetch real-time data on charge points realm.

## 🚀 Installation

```bash
npm install
```

## ▶️ Run locally

```bash
npm run dev
```

Browse **http://localhost:3001**

## ⚙️ Configuration

```env
NEXT_PUBLIC_API_URL=http://localhost:3000
```
